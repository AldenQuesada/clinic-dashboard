Fuzzy Search

Busca de pacientes usa pg_trgm com word_similarity (nГЈo similarity) вҖ” melhor pra casos onde a query Г© mais curta que o nome inteiro.

Algoritmo de score

score = GREATEST(
  word_similarity(query, name),
  0.95 if phone LIKE '%query_digits%',
  0.85 if name ILIKE '%query%'
)

WHERE (
  word_similarity(query, name) > 0.5
  OR name ILIKE '%query%'
  OR (query_digits != '' AND phone LIKE '%query_digits%')
)
